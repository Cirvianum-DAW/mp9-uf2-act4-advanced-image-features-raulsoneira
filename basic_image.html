Explanation about the width and height attributes:

The "width" and "height" attributes are used to control the dimensions of the image on the web page. These attributes do not change the actual size of the image file, but only affect how the image is displayed in the browser. If different dimensions are specified from the actual dimensions of the image, the image may be distorted or stretched to fit the specified values.
